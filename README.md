# ⌨️ KeyCodes Explorer 🔢

A fun and interactive **KeyCodes Explorer** built with **HTML**, **CSS**, and **JavaScript**.  
Press any key and watch it display its corresponding key code – a light-hearted and educational tool! 👩‍💻
